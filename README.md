
## Установка

 Создайте и активируйте виртуальную среду:
```удар
python -m venv venv
source venv/bin/activate # В Windows: venv\Scripts\activate
```

 Установите зависимости:
```удар
pip install PyMuPDF python-docx pdfplumber pdf2docx pytesseract opencv-python-headless numpy pandas
```

 Для поддержки оптического распознавания символов (отсканированных PDF-файлов) установите Tesseract:
`удар
# macOS
заваривайте установочный тессеракт

# Ubuntu/Debian
sudo apt-get install tesseract-ocr

# Окна
# Скачать с: https://github.com/UB-Mannheim/tesseract/wiki
```

## Использование

### Простое использование

```удар
# Конвертировать один PDF-файл (автоматически выбирается оптимальный метод)
python convert_pdf.py document.pdf

# Интерактивный режим
python convert_pdf.py
```

### Параметры командной строки

```удар
# Укажите выходной файл
python convert_pdf.py input.pdf -o output.docx

# Выберите метод преобразования
python convert_pdf.py document.pdf -m pdf2docx

# Пакетное преобразование всей папки
python convert_pdf.py --batch /путь/к/папке/с/pdf
```

### Python API

```питон
from comprehensive_pdf_converter import ComprehensivePDFConverter

# Создать конвертер
конвертер = ComprehensivePDFConverter()

# Конвертировать PDF
success = converter.convert('input.pdf', 'output.docx', method='hybrid')
```

## Методы преобразования

| Метод | Лучше всего подходит для | Скорость | Качество |
|--------|----------|-------|---------|
| гибридный | Большинство документов | Средний | Отличный |
| pdf2docx | Стандартные PDF-файлы | Быстро | Качественно |
| pymupdf | Сложные макеты | Средний | Хороший |
| pdfplumber | Документация с большим количеством таблиц | Медленная | Отлично подходит для таблиц |

## Устранение неполадок

### Общие проблемы

** Таблицы не распознаны должным образом**
   - Попробуйте метод `pdfplumber`
   - Используйте `comprehensive_pdf_converter.py` напрямую для большего контроля

** Отсканированный PDF-файл не конвертируется**
   - Убедитесь, что Tesseract установлен
   - Инструмент автоматически распознает отсканированные PDF-файлы и использует технологию оптического распознавания символов

** Проблемы с форматированием**
   - Попробуйте разные методы
   ` гибридный`- обычно дает наилучшие результаты

** Большие файлы**
   - Обработка может занять некоторое время
   - Следите за ходом выполнения в консоли

### Советы для достижения наилучших результатов

- По возможности используйте PDF-файлы высокого качества
- Для отсканированных документов важно обеспечить высокое качество сканирования
- Попробуйте разные способы, если первая попытка не увенчалась успехом
- Проверьте вывод консоли на наличие предупреждений о конкретных проблемах

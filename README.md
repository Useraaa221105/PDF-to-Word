# Комплексный конвертер PDF в Word

Мощный инструмент на Python, который преобразует PDF-файлы в документы Word с сохранением форматирования, таблиц, изображений и макета.

## Особенности

- ** Несколько методов преобразования**:
 - ** Гибридный **: Объединяет несколько библиотек для достижения наилучших результатов (рекомендуется)
 - **pdf2docx**: Быстрое преобразование с хорошим форматированием
 - **PyMuPDF**: Хорошо справляется со сложными документами
 - **pdfplumber**: Отличное извлечение таблиц

- ** Всесторонняя поддержка форматов**:
 - Текстовые PDF-файлы
 - Отсканированные PDF-файлы (с распознаванием текста)
 - Документы с большим объемом таблиц
 - Макеты с несколькими столбцами
 - Изображения и графика
 - Формы и аннотации
 - Зашифрованные PDF-файлы (если они доступны)

- Сохранение визуального форматирования**:
 - Стили шрифтов (жирный, курсив, подчеркивание)
 - Размеры и семейства шрифтов
 - Цвета и подсветка
 - Таблицы с объединенными ячейками
 - Расположение изображений
 - Разрывы страниц
 - Верхние и нижние колонтитулы
